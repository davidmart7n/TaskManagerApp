Project Details

TaskManagerApp

TaskManagerApp is an application designed by David Martín to manage and organize tasks efficiently. It allows users to create, track, and update tasks, with options for visualization, filtering, and organization by priority, due dates, and statuses.

Technologies Used

  • Java: Main programming language used for backend development.
  • Spring Boot: Framework for building the backend application, managing data, and connecting to the database.
  • Database (MySQL): Relational database used to store tasks, users, and task statuses.
  • API REST: Added but not implemented, allowing potential future integration with external systems or modern frontends.
  • Thymeleaf: Template engine for generating dynamic HTML pages and connecting the backend with the frontend.
  • HTML5 and CSS3: Structuring and visual design of the user interface.
  • Bootstrap 5: Framework for creating a responsive interface that adapts to different devices.
  • JavaScript: For dynamic interactions, form validation, and real-time updates.
  • Git: Version control system for collaborative and secure code development.

Authors's Objectives

The objective of this project is to create a simple task management system to put my programming skills into practice and showcase it as part of my personal portfolio.
